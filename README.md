## 🏷️ 1. Project Title
**CodeSync – Code Together, Sync Instantly**

---

## 🧩 2. Description  

**CodeSync** is a remote interview and collaborative coding platform built using the **MERN stack (MongoDB, Express, React, Node.js)**.  

It allows users to:
- Conduct **live coding interviews**
- **Collaborate in real-time**
- Practice coding problems independently  

The platform integrates:
- Real-time video and chat  
- **Collaborative code editor (multi-user editing simultaneously)**  
- Coding problem library  
- Live session dashboard  
- **AI-powered assistance using Gemini API**  

---

## ⚙️ 3. Features  

| Feature | Description |
|----------|--------------|
| 👥 **Real-Time Collaboration** | Live interviews with video, chat, and shared editor |
| 💻 **Collaborative Code Editor** | Multiple users can edit code simultaneously |
| 🤖 **AI Assistance** | Code generation & explanation using Gemini |
| 💬 **Chat Integration** | Real-time messaging |
| 📚 **Practice Problems** | Structured coding problems |
| 📊 **Dashboard** | Track sessions & activity |
| 🔐 **Authentication** | Clerk + JWT |
| 🎨 **Modern UI** | Tailwind CSS + DaisyUI |
| ⚡ **Real-Time Sync** | Stream API + Socket.io |
